Um beim Umschalten der Szenen auch die Umschalte Animation zu nutzen musst Du Autohotkey Installieren. (https://autohotkey.com/download/)

Nach der Installation starte die Autohotkey Datei obs.ahk aus dem Overlay Verzeichniss.

Folgende Hotkeys müssen im OBS-Studio Eingestellt sein, damit der Szenenwechsel Funktioniert (Ist bei Installation des Overlays Automatisch der Fall):
	Ctrl + Alt + Numpad1 (UND zusätzlich Ctrl + Alt + F9)  (Zu Szene Main Wechseln)
	Ctrl + Alt + Numpad2 (UND zusätzlich Ctrl + Alt + F10) (Zu Szene Ingame Wechseln)
	Ctrl + Alt + Numpad3 (UND zusätzlich Ctrl + Alt + F11) (Zu Szene Idle Wechseln)
	(Diese sind bei der Standartinstallation von OBS voreingestellt!)

Weiterhin muss in den Szenen Main, Ingame und Idle die Quelle "Übergang" an Oberster Stelle Stehen!

Nachdem das AHK Script gestartet ist, muss es noch initialisiert werden. Aktiviere hierzu das OBS Studio Fenster und bewege die Maus über die Quelle "Übergang"
Benutze anschließend (Ctrl + Shift + Numpad0) ODER (Ctrl + Shift + F12) um das Script zu Initialisieren. Du erhälst ein Kurzes Bestätigungsfenster.

Danach kannst du mittels Ctrl + Shift + Numpad0 bis Numpad3 bzw. Ctrl + Shift + F9 bis F11 Zwischen den Szenen umschalten mit aktivem Übergang.

Du Solltest das Spiel welches Du streamst idealerweise auf Borderless Window gestellt haben, das du sonst verlängerte Wartezeiten beim Umschalten der Szene hast, da das Spiel kurzzeitig minimiert werden muss!
